# Система учёта обращений клиентов

Учебный проект по дисциплине «Разработка web-приложений на языке JavaScript».

Проект представляет собой заготовку web-приложения для регистрации и отслеживания обращений клиентов в организацию. В этой лабораторной работе выполнены базовая структура проекта, простая верстка и подключение JavaScript.
